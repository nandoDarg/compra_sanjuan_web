1) Objetivo del producto
- Marketplace local (San Juan) para publicar, explorar y contactar vendedores.
- Flujo principal: feed publico -> detalle -> contacto WhatsApp.
- Rutas privadas: crear/editar/mis publicaciones.

- React 19.2.4 + TypeScript
- Supabase (Auth, Postgres, Storage)
- Tailwind CSS v4
- PostHog analytics
- react-easy-crop para recorte de imagenes

9) Script de seed (resumen de uso)

- Inserta >= 40 publicaciones realistas.
- Sube 2-4 imagenes por publicacion desde Unsplash al bucket post-images.
- Inserta vehicle_details para categorias vehiculares.
- Variables requeridas:
  - SUPABASE_URL (o NEXT_PUBLIC_SUPABASE_URL)
  - SUPABASE_SERVICE_ROLE_KEY
  - UNSPLASH_ACCESS_KEY
- Ejecucion:
  - npx tsx scripts/seed-posts.ts
