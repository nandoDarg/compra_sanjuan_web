# Vision de producto

tratohechoSJ busca ser el marketplace local mas simple para publicar, descubrir y contactar vendedores en segundos.

## Norte de producto

- Explorar sin friccion (sin exigir login para mirar).
- Publicar rapido desde mobile.
- Convertir contacto en WhatsApp con el minimo de pasos.

## Enfoque actual

El proyecto ya no esta en etapa de prototipo tecnico.
El foco es validacion de uso real con usuarios reales.

Prioridad inmediata:

- adquirir usuarios
- medir comportamiento
- corregir fricciones UX detectadas en campo

## Principios de decision

- Primero validacion, despues complejidad.
- Una sola base de codigo.
- Cambios guiados por uso real, no por ideas abstractas.

## Metricas iniciales sugeridas

- publicaciones creadas por semana
- aperturas de detalle
- clics en WhatsApp
- tasa de abandono en create-post
- tiempo promedio hasta primer contacto
